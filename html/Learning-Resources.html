Been to a RoboJam?
Want to learn more?

There are lots of sites out there with some fantastic resources for learning.  We couldn't list them all, but here are some particularly useful ones to help you get started if you want to build your very own robot or just learn more about coding!  

And parents - we hear you ask: "Why is coding important?" 

Inexpensive robot kits are available from the CamJam site, including worksheets on how to make them work.  Don't forget you'll also need a Raspberry Pi to drive it.

Of course all our robots are powered by Raspberry Pi computers.  There is no end to the learning possibilities (and fun) you can have with one of these.  Find out more on the Raspberry Pi website.
